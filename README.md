# LA1 AI Entertainment System

## Structure
- `frontend/` - Next.js 14 frontend (deploy to Vercel or Railway)
- `backend/` - Express.js backend (deploy to Railway)

## Features
- Member system (register/login)
- Dashboard with balance
- Crypto deposit (USDT TRC20 via NOWPayments)
- Telegram notifications
- Risk control framework

## Deploy
1. Backend: Railway (root directory: backend/)
2. Frontend: Vercel or Railway (root directory: frontend/)
